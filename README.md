# 2️⃣ Customer Shopping Behavior Analysis

## 📌 Description
This project analyzes **3,900 customer transactions** to understand **shopping behavior, customer spending patterns, product preferences, and subscription trends.**

The goal is to generate **business insights that help companies improve marketing strategies and customer retention.**

## 📊 Key Insights
- Revenue comparison between **male and female customers**
- Spending behavior of **subscribers vs non-subscribers**
- Identification of **top rated products**
- Impact of **discounts on purchase behavior**
- **Customer segmentation** into New, Returning, and Loyal customers
- Revenue contribution by **different age groups**

## 🛠 Tools Used
- Python (Pandas)
- PostgreSQL
- SQL
- Power BI
- Data Visualization
